# Antique Atlas 4 - Still Life Biomes Support

Adds basic visual map support for Still Life biomes to [Antique Atlas 4](https://modrinth.com/mod/antique-atlas-4).

### ⚠️ Please Note (Work in Progress)
This resource pack was put together quickly to make the biomes playable and mapped as soon as possible. Because of this, **you should expect some inaccuracies or just straight-up wrong textures**. It is a work in progress and will likely be updated in the future to improve visual fidelity and accuracy.  
Testing how all biomes look and which textures to use takes a lot of time and effort.  

*Be aware that texture changes will only apply to newly explored areas. It won't automatically make the '???' texture disappear from already mapped areas (Don`t know if there is a way to delete scanend portions)*

If you come up with an improvement or face a bug, feel free to contribute or open an issue!

---

##  Compatibility & Versions
* **It should work** with the most recent versions of Still Life available on CurseForge/Modrinth.
* It was mainly made and tested using the **[Patreon](https://patreon.com/Jayzet) version**. 

---

##  Installation

1. Click the green **Code** button at the top of this repo and select **Download ZIP**.
2. Place the `.zip` file into your Minecraft `resourcepacks` folder.
3. Activate the pack in-game.
